## 升级方案：五层融合生成法

将模块四拆为五个独立子模块，每层用不同的 Prompt 策略，最后融合。

### 子模块 4.1：价值观层 ⭐
**定位**：所有创作的基石。先确定「这个剧本要传递什么」，再谈怎么写。

| 项目 | 内容 |
|------|------|
| **输入** | 采访素材 + 客户确认的核心价值观 |
| **AI 策略** | 独立 Prompt，专门提炼价值观主张 |
| **产出** | ① 价值观金句（3-5句） ② 价值观在每集的位置映射表 |
| **Prompt 重点** | 对标经典电影价值观，不是喊口号，是人物行动中自然流露 |

### 子模块 4.2：视角与设定层 🔭
**定位**：找到一个独特的切入角度，让这个故事区别于同类。

视角类型参考：技术视角 / 人性视角 / 时代视角 / 家庭视角

### 子模块 4.3：节奏与算法层 📊
**定位**：确保符合短剧平台的算法规则——爽点密度、钩子位置、情绪曲线。

**热点注入**：每月更新趋势，自动调整钩子密度和开场策略。

### 子模块 4.4：台词与信息密度层 ✍️
**定位**：确保每句对白有信息增量，拒绝「你吃了吗」型废话。

台词功能分类：【推进】【揭示】【渲染】【钩子】【金句⭐】

### 子模块 4.5：情绪引导层 🎭
**定位**：精确控制观众的情绪体验——什么时候笑、什么时候哭、什么时候拍大腿。

情绪设计原则：铺垫→引爆→释放，情绪对比 > 单一情绪

## 融合流程
```
采访素材 → [4.1价值观] → [4.2视角] → [4.3节奏+热点] → [4.4台词] → [4.5情绪审查] → 完整剧本
```

## 与 v1.0 的差异
| 维度 | v1.0 | v2.0 |
|------|------|------|
| 生成方式 | 单次 Prompt | 五层分步 + 融合 |
| 质量可控性 | 低（黑箱） | 高（每层独立调优） |
| 热点响应 | 无 | 节奏层每月注入 |
| 台词质量 | 参差不齐 | 每句功能标注 |
| 情绪设计 | AI 直觉 | 精确到每集情绪目标 |
