## 一、整体流程概览

```
用户访谈(19题) → 故事架构师 → 人物设计师 → 剧本写手 → 台词润色师 → 剧本评估师 → 总编审 → 完整剧本
```

| 阶段 | 作用 | 输入 | 输出 |
|------|------|------|------|
| 用户访谈 | 收集真实素材 | 用户回答 | 19 条深度回答 |
| ① 故事架构师 | 提炼冲突结构 | 用户回答 | 一句话故事、三幕结构、核心冲突、人物种子、情绪曲线 |
| ② 人物设计师 | 塑造角色弧光 | 架构分析 + 用户回答 | 主角档案、配角矩阵、人物关系图、角色声音样本 |
| ③ 剧本写手 | 创作完整剧本 | 架构 + 人物 + 用户回答 | 完整剧本（含分集大纲、剧本样章、金句、商业建议） |
| ④ 台词润色师 | 优化对白质感 | 剧本初稿 | 润色后的剧本（潜台词增强、角色辨识度提升） |
| ⑤ 剧本评估师 | 深度评估 | 完整剧本 + 人物设计 | 八维评分 + 三密度评分 + 审查合规报告 |
| ⑥ 总编审 | 最终裁定 | 全部产出 | GREENLIGHT/YELLOW/RED 裁定 + 导演笔记 + 平台建议 |

## 二、用户访谈体系

### 2.1 访谈结构（4 层 × 19 题）

以一个创业者的「传奇之路」方向为例：

**第 1 层：起点（5 题）**
```
Q1: 你的创业故事从哪里开始？那一年的你，是什么状态？
Q2: 最初让你决定创业的那个瞬间是什么？
Q3: 创业之前，你做什么工作？那段经历给了你什么？
Q4: 最早和你一起出发的人是谁？现在还在身边吗？
Q5: 如果用三个词形容当时的你，是哪三个？
```

**第 2 层：转折（6 题）**
```
Q6: 创业至今，最艰难的一次抉择是什么？
Q7: 你的「至暗时刻」发生了什么？那段时间你靠什么撑过来？
Q8: 有没有哪个人在你最困难的时候帮了你？具体做了什么？
Q9: 第一次明确感到「我赢了」的那个瞬间是什么？
Q10: 让你印象最深的一次背叛或失望是什么？
Q11: 如果有竞争对手，他们是怎么对你的？你如何回应？
```

**第 3 层：情感（5 题）**
```
Q12: 创业对你家庭/亲密关系的影响是什么？
Q13: 你最愧疚的一件事是什么？
Q14: 如果回到创业第一天，你会对自己说什么？
Q15: 有哪一个瞬间让你觉得「这一切都值得」？
Q16: 有没有一次你想放弃但没有放弃的时刻？为什么？
```

**第 4 层：精神（3 题）**
```
Q17: 你希望通过这个故事传递给观众什么价值观？
Q18: 你认为你身上最独特的精神是什么？
Q19: 如果这个故事是一部电影，你希望它在哪些场景让你落泪？
```

### 2.2 追问机制

每道题回答后，AI 可根据回答内容生成 1 个追问（不超过 25 字），聚焦回答中"没说透的点"。追问原则：
- 自然，像朋友聊天，不要像面试官
- 聚焦对方回答中最有意思但没说透的那个点
- 问"什么时候""在哪""谁""怎么做的"而非"为什么"

```
追问 Prompt：
你是一个专业的采访者。用户刚刚回答了一个关于自己经历的问题。
请根据他的回答，提出一个简短的、有洞察力的追问（不超过25个字），
帮助挖掘更具体的细节或情感。直接返回纯 JSON：{"followUp":"你的追问"}
```

## 三、六 Agent 流水线

### 3.1 ① 故事架构师

**角色**：从真实人生经历中精准提炼戏剧冲突。

**核心任务**：
1. 一句话故事（20 字内，须有情感钩子/认知反差）
2. 故事梗概（300 字，起因→转折→高潮→结局）
3. 三幕结构（每幕标注关键事件、情绪基调、时长占比）
4. 核心冲突（人vs人/人vs环境/人vs自我，含升级路径）
5. 主题提炼（一句话核心价值观）
6. 人物种子（3-5 个可戏剧化的角色雏形）
7. 情绪曲线（5-7 个关键情绪节点）
8. 金句灵感（从用户原话中提取 3 句金句胚子）

**分析原则**：
- 必须基于用户真实回答，不可虚构重大事件
- 优先识别「至暗时刻」和「高光瞬间」
- 冲突必须有层次，不只是「困难→克服」
- 人物必须复杂：主角有恐惧和软肋
- 金句必须从用户原话中寻找灵感

[▶ 完整 Prompt 模板见第十节](#101-故事架构师-system-prompt)

---

### 3.2 ② 人物设计师

**角色**：将真实人物转化为有魅力、有弧光、有辨识度的戏剧角色。

**核心设计理念**：
1. **真实大于完美** — 角色的魅力和弱点必须基于真实人物
2. **"想要"和"需要"分离** — 想要是表面的（钱、权力、认可），需要是内在的（被爱、自证、解脱）
3. **配角法则** — 每个配角必须有独立动机弧线（贵人/镜子/催化剂/对手）
4. **语言指纹** — 每个角色有独特说话方式（口头禅、句式偏好）
5. **关系张力** — 角色间有"未说出口的话"

**输出内容**：
- 主角完整档案（外在/内在驱动/恐惧/软肋/语言风格/成长弧光详述）
- 配角矩阵（3-5 人，含功能、关系动态、独立弧线）
- 对手/困境深化（合理动机，让观众共情）
- 人物关系图谱（盟友/对手/亦敌亦友/师徒）
- 角色声音样本（每个角色一句标志性台词）

[▶ 完整 Prompt 模板见第十节](#102-人物设计师-system-prompt)

---

### 3.3 ③ 剧本写手

**角色**：将故事架构和人物设计落地为可拍摄的短剧剧本。

**核心创作标准**：
1. 经典电影的价值观 — 传递普世、深刻的价值主张
2. 独特的切入视角 — 与众不同的叙事角度
3. 短剧算法友好的开场和爽点 — 前三秒强冲突/大反差/悬念
4. 高密度高信息量台词 — 每句对白有信息增量
5. 全程引导观众情绪 — 精确设计情绪曲线

**创作铁律（2026）**：
- 开局身份反转/强冲突是最高点击率手法
- 爽点密度：平均每 90 秒一个微爽点
- 付费钩子必须狠：每集倒数第 3 句制造悬念
- 情绪对比法：极悲→极喜的反差
- 金句密度：每集至少 2-3 句可截图传播的台词
- 商战+情感双线模式优先
- 禁止大段回忆（闪回不超过 20 秒）
- 纯对话推进每段不超过 4 句，必须配行动

**三密度要求**：详见[第四节](#四三密度注入标准)。

[▶ 完整 Prompt 模板见第十节](#103-剧本写手-system-prompt)

---

### 3.4 ④ 台词润色师

**角色**：让对白从"能看"变成"精彩"。不改结构、不改情节，只优化台词。

**五大润色铁律**：

| 法则 | 说明 | 示例 |
|------|------|------|
| 潜台词法则 | 角色说的和想的不一样 | ❌"我很生气" → ✅"行，挺好的"（转身背对） |
| 信息密度法则 | 每句台词必须有信息增量 | ❌"我们营收5000万" → ✅"5000万？够给老周发工资的" |
| 角色语音指纹 | 去角色名只看台词，能分辨是谁 | 老板=短句+命令式，技术宅=细节控+术语 |
| 金句爆破 | 情绪高点必须有传播力台词 | ❌"创业要努力" → ✅"死磕到所有人都觉得你疯了，就对了" |
| 去脸谱化 | 反派不说反派的话 | ❌"哈哈哈你们完了" → ✅"这条路我走过，不是为难你" |

[▶ 完整 Prompt 模板见第十节](#104-台词润色师-system-prompt)

---

### 3.5 ⑤ 剧本评估师

**角色**：用锐利眼光审视剧本，给出冷酷但建设性的反馈。不说"挺好的"，只说"哪里不好、为什么、怎么改"。

**评估维度**：详见[第六节](#六八维评估体系)。

**附加检查**：
- 三密度评分（情绪/信息/情节各自打分+等级）
- 审查红线扫描（A类 7 项 + B类 6 项）
- 最强 3 优点 / 最弱 3 短板
- "如果只能改一处"的最高 ROI 建议

[▶ 完整 Prompt 模板见第十节](#105-剧本评估师-system-prompt)

---

### 3.6 ⑥ 总编审（Showrunner）

**角色**：看完所有产出后说"过"或"重写"的人。

**五大裁定维度**：
1. 整体一致性审查 — 故事/人物/对白/评估之间有无矛盾
2. 导演笔记 — 对全剧组说的第一句话（定调）
3. 核心问题诊断 — "不删改 XXX，这片子没法拍"
4. 市场判断 — 竞品分析 + 差异化建议 + 双平台适配
5. 最终裁定 — GREENLIGHT（可拍）/ YELLOW（修改后重审）/ RED（不建议）

## 四、三密度注入标准

### 4.1 情绪密度

**定义**：单位时间内观众情绪被调动的次数和强度。

| 级别 | 标准 | 适用场景 |
|------|------|----------|
| 极 | 每 5-8 秒 1 次情绪波动 | 黄金三秒开场、最终对决 |
| 高 | 每 15 秒 1 次情绪波动 | 高潮、付费卡点前 |
| 中 | 每 30 秒 1 次情绪波动 | 常规推进 |
| 低 | 每 60 秒 1 次情绪波动 | 过场、交代背景 |

**情绪密度公式**：
```
情绪密度分 = (情绪波动次数 / 场景秒数 × 60) × 情绪强度系数
情绪强度系数：燃/爽 = 1.2，虐 = 1.0，暖 = 0.8，悬 = 0.9
```

**池子法则**：
- 单一场景内情绪类型不超过 2 种（太多会乱）
- 相邻场景的情绪必须有对比（燃→虐→燃，不能燃→燃→燃）
- 每集至少一次情绪"过山车"（低谷→高峰→回落）
- 禁止连续 15 秒以上无情绪变化的平铺段落

### 4.2 信息密度

**定义**：单位台词内传递的新信息量。废信息 = 观众已知的重复内容。

**信息价值排序**：
```
揭示关系/颠覆认知 ⭐⭐⭐⭐⭐ > 推进冲突 ⭐⭐⭐⭐ > 暗示伏笔 ⭐⭐⭐⭐ > 塑造人设 ⭐⭐⭐ > 交代背景 ⭐⭐ > 废话寒暄 ☆
```

**每集标准**：
- 第 1 集：15-20 条新信息（人设+世界观+冲突种子）
- 第 2-5 集：8-12 条新信息
- 中间集：5-8 条新信息
- 付费卡点集：10-15 条新信息（加量）

**铁律**：
- 每条对白必须至少携带一条新信息
- 禁止角色互讲观众已知信息
- 禁止纯寒暄对白（"你好""请坐""谢谢"）

**自检法**：`删掉这句话，观众还能理解剧情吗？→能→删除`

### 4.3 情节密度

**定义**：单位时间内情节推进步数。一步 = 一个不可逆的状态变化。

**推进阶梯**：
```
微转折（每 30 秒）→ 小反转（每 3-5 集）→ 中高潮（每 10 集）→ 大逆转（全剧 2-3 次）
```

**每集标准**：
- 第 1 集：至少 3 个微转折 + 1 个悬念钩子
- 常规集：至少 2 个微转折 + 1 个钩子
- 付费卡点集：至少 4 个微转折 + 2 个钩子（前中后各挂一次）

**自检法**：`删掉这个场景，下个场景还成立吗？→成立→可删`

### 4.4 三密度总分

```
三密度总分 = 情绪密度分 × 0.4 + 信息密度分 × 0.3 + 情节密度分 × 0.3

优（8-10分）：三密度均在「高」以上
良（6-7分）：两个密度在「高」以上
中（4-5分）：一个密度在「高」以上
差（<4分）：三密度均在「中」以下 → 必须重写
```

### 4.5 注入指令（可直接复制到 System Prompt）

```
# 📊 三密度要求
本场景时长约 XX 秒，要求至少 X 次情绪波动。
请在以下位置设计情绪引爆点：
- 开场第 3-5 秒：第一次情绪刺激
- 中段第 XX 秒：情绪压制/反转
- 结尾前 5 秒：情绪爆发/悬而未决
禁止：连续 15 秒以上无情绪变化的平铺段落。
禁止：纯寒暄对白、角色互讲已知信息。
每条对白必须至少携带一条新信息。
每写完一集自检：
1. 最近 15 秒内观众的情绪有变化吗？
2. 删掉这句话，观众还能理解剧情吗？（能→删除）
3. 删掉这个场景，下个场景还成立吗？（成立→删除）
```

## 五、开场自洽闭环要求

黄金开场炸裂后，第 4-10 集必须完成"自洽闭环"——解释为什么开场那么炸。

### 自洽三问

1. **动机自洽**：开场角色的反常行为，后面有没有合理的动机解释？
2. **信息自洽**：开场暗示的悬念，后面有没有对应的真相揭示？
3. **逻辑自洽**：开场建立的规则/设定，后面有没有被推翻或矛盾？

### 自洽验证时间线

```
第 1-3 集：抛出炸裂开场 → 建立悬念
第 4-6 集：给出第一次解释（部分真相）→ 但引出更大疑问
第 7-10 集：完全自洽闭环 → 观众恍然大悟
```

### 反例与正例

- ❌ 开场「我给自己烧纸」→ 后面从未解释 → 炸完就跑，观众愤怒
- ✅ 开场「我给自己烧纸」→ 第 4 集揭示为引蛇出洞 → 第 8 集真凶现身 → 自洽闭环

### 注入指令

```
# 开场自洽要求
当前开场：[开场文案]
请在剧中标注：
- 第 4 集：第一次自洽揭示（部分真相）
- 第 7 集：第二次自洽揭示（更大真相）
- 第 10 集：完全自洽闭环（所有谜底揭晓）
铁律：开场提出的每一个反常行为，在后续剧集中都必须有对应解释。
```

## 六、八维评估体系

| 维度 | 权重 | 10 分标准 | 扣分项 | 自检法 |
|------|------|-----------|--------|--------|
| 黄金开场 | **1.5** | 前 3 句必须有冲突/悬念/反差 | 慢热铺垫、大段背景介绍 | 遮住只看前三行，看完还想继续吗？ |
| 节奏设计 | **1.3** | 每 30 秒一个微转折，爽点均匀分布 | 注水段落、连续 3 分钟无情绪变化 | 画情绪曲线，有没有平直段超过 30 秒？ |
| 情绪价值 | **1.2** | 爽点让人拍大腿，泪点让人鼻子酸 | 起伏不足、哭点没铺垫 | 看完什么感觉？"没感觉"= 不及格 |
| 悬念钩子 | **1.4** | 每集末尾钩子精准狠，必须点下一集 | 平淡结尾、钩子可预测 | 每集最后三句——能忍住不点下一集吗？ |
| 大纲结构 | **1.0** | 奠基→成长→危机→逆袭→升华，逻辑自洽 | 因果断裂、为冲突而冲突 | 抽掉任何一集，故事还成立吗？ |
| 台词质量 | **1.1** | 每句有信息增量，去角色名能分辨谁 | "你吃了吗"类无信息对白、风格雷同 | 遮住角色名读台词，能分辨谁在说吗？ |
| 人物成长 | **1.0** | 主角从 A 到 B 转变清晰可信 | 主角没真正改变、转变太快 | 第一集 vs 最后一集的主角真的变了吗？ |
| 逻辑纠错 | **0.8** | 时间线严谨、因果关系通顺 | 时间线矛盾、行业细节穿帮 | 列出所有"因为…所以…"，有无逻辑断裂？ |

### 加权评分公式

```
加权总分 = goldOpening × 1.5 + rhythmDesign × 1.3 + emotionalValue × 1.2
         + suspenseHook × 1.4 + outlineStructure × 1.0 + dialogueQuality × 1.1
         + characterGrowth × 1.0 + logicCheck × 0.8

满分 80，评级标准：
S+（70-80）可直接拍摄上线
A（60-69）微调后可上线
B（50-59）需较大修改

### 8.1 核心差异

| 维度 | 抖音 | 红果 |
|------|------|------|
| 用户进入方式 | 刷到的，被动 | 搜到的/推送的，半主动 |

| 集长 | 1-2 分钟 | 2-5 分钟 |
| 付费点（30 集） | 第 8、15、25 集 | 第 10、18、25 集 |
| 付费点（50 集） | 第 10、20、35、45 集 | 第 12、25、40、48 集 |
| 付费点（80 集） | 第 12、25、40、55、70 集 | 第 15、30、45、60、75 集 |
| 风格偏好 | 强冲突、高反转、快节奏 | 可有一点铺垫，情绪层次更丰富 |
| 完播率关键 | 前 3 秒 + 每 30 秒一个爽点 | 前 5 秒 + 每 60 秒一个爽点 |
| 评论区引导 | 极其重要 | 次重要 |

### 8.2 开场差异化

```
⚡ 抖音开场（第 1 集）：
- 0-3 秒：身份锁定 + 反常识宣言 + 悬念炸弹
- 3-15 秒：快速推进冲突
- 15-30 秒：第一次微反转
- 30-60 秒：建立付费期待
- 结尾：强钩子

⚡ 红果开场（第 1 集）：
- 0-3 秒：同上
- 3-30 秒：稍多铺垫，给情绪层次

- 结尾：中等钩子

### 8.3 台词风格差异

```
抖音台词：更口语、更网络、更"喊出来"

红果台词：可稍书面、可更有层次
  ✅ "他等了十年，等的就是今天这一刻。"
  ✅ "你以为赢了？其实从一开始，你就输了。"

总分 = 情绪×0.4 + 信息×0.3 + 情节×0.3

## 附录：调用顺序建议

如果你想在其他大模型中手动按本 SOP 创作，推荐以下调用顺序：

```
第 1 轮：粘贴「10.1 故事架构师 Prompt」+ 用户回答 → 获得架构分析
